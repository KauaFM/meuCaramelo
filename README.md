# 🐾 meuCaramelo

Landing page desenvolvida com foco em adoção de pets, com layout moderno e responsivo.

## 🚀 Tecnologias

- React
- Vite
- Tailwind CSS

## 🖥️ Projeto Online

Acesse o site:[(https://meu-caramelo-dmdk.vercel.app/)]

```bash
